For "All Configurations"

VC++ Directories > Include Directories: Add include directory
VC++ Directories > Library Directories: Add library directory
Linker > Input > Additional Dependencies: Add SDL2_image.lib

In main() add the lines:

#include <SDL_image.h>

IMG_Init(IMG_INIT_JPG);		//Just after SDL_Init(SDL_INIT_VIDEO);



SDL_Surface* image = IMG_Load("image.jpg");



IMG_Quit();			//Just before SDL_Quit();
